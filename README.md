# 🌍 World Clock — Team Timezones

A beautiful, responsive world clock with five different visualizations:

- **Flip Clock** — retro split-flap display
- **Analog Wall** — classic clock faces
- **Globe** — rotating Earth with day/night terminator
- **Map** — NASA Earth at Night with timezone pins
- **Horizon Strip** — 24-hour daylight bar

## Live Demo

👉 [imhickey.github.io/world-clock](https://imhickey.github.io/world-clock/)

## Features

- Six world timezones (US West, US East, Dublin, Israel, India, Sydney)
- Day/night indicators on all views
- Seconds toggle
- Keyboard shortcuts (1–5 to switch views)
- Fully responsive — works on desktop, tablet, and phone
- View preference saved to localStorage
- Single self-contained HTML file — no build step
